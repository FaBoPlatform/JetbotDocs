# SDカードの作成

## SDカードの焼き込み

Jetbot用のSDカードをダウンロードしてきて、[Etcher](https://www.balena.io/etcher/)でSDカードに焼き込む。

[SDカードのイメージ](https://drive.google.com/file/d/1RgQ99QOqhcNxivSNJpetXdoOCqUWAWH_/view)

!!! Tip
	SDカードは、64GB以上の容量が必要となります。

## JetBotの起動

SDカードをJetson Nanoに差し込み起動する。

|項目|内容|
|:--|:--|
|ID|jetbot|
|Pass|jetbot|
