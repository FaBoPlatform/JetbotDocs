# 2.Object Following

Object Followingは、学習済みモデルを使ったデモのみサンプルになります。

|JetPack|Download|
|:--|:--|

## Notebookの実行

Object FollowingのNotebookは、データセット作成はおこなわず、学習済みモデルでのデモになります。

デモでは、選択したラベルの物体を追従して、後を追って走ります。
