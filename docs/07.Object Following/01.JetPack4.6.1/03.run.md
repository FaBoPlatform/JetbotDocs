# 3.自動走行

## 自動走行(02_live_demo_JP.ipynb)

自動走行では、直進できる場合に物体を追従するように走行します。  
物理メモリ不足の影響でOpenCVカメラ画像取得処理が落ちてしまうことがあります。  
1-2分待ってもセルが実行中のままだったり、エラーになったりする場合はJetBotの電源を一度落としてから、再起動してください。
