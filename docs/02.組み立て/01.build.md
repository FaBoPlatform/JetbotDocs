## 5. Wifi Moduleの取り付け

Wifi Moduleを取り付けるために、Jetson Nano moduleを取り外します。

![](./img/kit015.JPG)

![](./img/kit016.JPG)

![](./img/kit017.JPG)

Wifi Moduleを取り付けるために、真ん中のネジを外しておきます。

![](./img/kit018.JPG)

Wifi Moduleは、8265NGWもしくは、8265NGWMGを取り付けます。

![](./img/kit019.JPG)

![](./img/kit020.JPG)

![](./img/kit021.JPG)

![](./img/kit021_2.JPG)
