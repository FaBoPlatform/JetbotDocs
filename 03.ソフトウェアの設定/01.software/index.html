    ソフトウェアの設定

    FaBo JetBotに対応させたRepoをGit cloneし、インストールする。

    FaBo PCA9685のインストール

    1
2
    $ git clone https://github.com/FaBoPlatform/FaBoPWM-PCA9685-Python
$ pip3 install FaBoPWM-PCA9685-Python/

    JetbotのFaBo Support

    1
2
3
4
    $ git clone https://github.com/FaBoPlatform/jetbot
$ cd jetbot
$ git checkout FaBo
$ sudo python3 setup.py install
