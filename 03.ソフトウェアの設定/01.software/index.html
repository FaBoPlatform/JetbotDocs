    Terminalの起動

    最新版のjetbotプロジェクトをインストールします。

    cd ~/
git clone https://github.com/NVIDIA-AI-IOT/jetbot

    最新版のJetbotプロジェクトのnotebookを~/Notebooksフォルダにrsyncします。

    1
2

    cd ~/
sudo apt-get install rsync
rsync -r jetbot/notebooks ~/Notebooks
